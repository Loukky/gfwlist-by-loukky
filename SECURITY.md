# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

Instead, use the private vulnerability reporting feature provided by GitHub:

- Go to the [Security and Quality](https://github.com/Loukky/gfwlist-by-loukky/security) tab.
- Click the `Report a Vulnerability` button to open a private security advisory draft.

Do NOT open a public issue.

We will respond as soon as possible.
